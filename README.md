# Fire Incident Recording System (FIRS)

## Background
Fire incident reporting in Cagayan de Oro City was traditionally done manually, leading to delays, errors, and difficulty in organizing records. The Fire Incident Recording System (FIRS) was developed to replace paper-based logging with a simple and efficient digital solution for internal use.

---

## Overview
FIRS is a web-based application used by fire personnel to record, manage, and retrieve fire incident data. It centralizes all records in one system, making it easier to track incidents, maintain accuracy, and generate reports when needed.

---

## System Description (How It Works)
Users log in using a shared account and manually enter fire incident details or upload data in bulk using Excel or CSV files. The system validates the input, stores the records, and automatically arranges them in chronological order.

Users can search and filter incidents based on date, location, or keywords. The system automatically computes totals such as number of incidents, damage, injuries, and casualties.

Reports can be generated and exported to Excel, including filtered data or annual summaries. All records remain organized, standardized, and easily accessible within the system.
